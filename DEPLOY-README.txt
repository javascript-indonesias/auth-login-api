** PETUNJUK DEPLOY BUNDLE **

1. Copy folder /bundle ke server.
1. Perbarui parameter di .env yang dikhususkan untuk jalan di server produksi.
2. Jalankan NPM Install untuk memasang package eksternal atau dependencies Node JS yang dibutuhkan oleh aplikasi server.
3. Setelah memasang semua dependency node_modules, jalankan file "server.bundle.js" dengan Forever JS , PM2, atau Nodemon.
